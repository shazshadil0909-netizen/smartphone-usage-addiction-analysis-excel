Smartphone Usage & Addiction Analysis Dashboard (Excel)
About the Project

This project is an interactive Excel dashboard created to analyze smartphone usage and addiction patterns. It focuses on how users spend time on their smartphones and explores relationships between screen time, sleep, gaming, social media usage, and addiction levels.

The dashboard was built using Microsoft Excel and is designed to present the data in a simple, interactive, and easy-to-understand format.

Project Objective

The main objectives of this project are to:

Analyze smartphone usage patterns.
Understand how screen time changes across different addiction levels.
Compare time spent on social media and gaming.
Explore the relationship between screen time and sleep duration.
Present insights through an interactive Excel dashboard.

Key Findings
Screen time increases with addiction level

Users with higher addiction levels spend more time on their smartphones.

Addiction Level	Average Screen Time
Mild	5.46 hrs
Moderate	8.38 hrs
Severe	8.70 hrs
Most users fall into the Moderate addiction category
Mild – 21%
Moderate – 42%
Severe – 37%
Social media usage is higher than gaming

Across all addiction levels, users spend more time on social media than gaming.

Weekend screen time is higher than weekday screen time

Users generally spend more time on their phones during weekends.

Sleep hours remain relatively stable

Within this dataset, average sleep duration does not change significantly as screen time increases.

Note: This observation is based only on users aged 18–32 years in this dataset and should not be generalized to other age groups.
